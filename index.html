How to Apply CSS to HTML

To include an external CSS file in an HTML file, use the <link> tag: <link rel="stylesheet" href="style.css">

How to Use JavaScript in a Webpage

To link to an external JavaScript file, use the <script src="..."> tag. You can also include inline JS: <script>
window.addEventListener("load", () => {
console.log("Page is fully loaded");
});
</script>
